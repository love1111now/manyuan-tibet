# 滿願藏庫｜設計與內容方針（Design Notes）

> 目標：高美感 + 高說服力 + 高轉換率（導向綠界付款）+ 全面 SEO。

## 1) 核心敘事（以《金光明最勝王經》為主軸）
- **危機（先讓人點頭）**：不是你不努力，而是「福德資糧不足、財庫漏損、決策迷霧、關係失位」在長期消耗。
- **證據（讓人相信）**：以《金光明最勝王經》的「護國、增益、除障」意象作為論述中軸，主張「如法供養＋回向」是可被理解的心行與行動路徑。
- **解法（讓人看見下一步）**：四尊四條路徑（穩財／守財／清明／貴人），每尊皆提供清楚的「適用情境 → 方案 → 流程 → 常見疑慮 → 行動 CTA」。
- **行動（讓人立刻成交）**：每一頁都要有明確 CTA：
  1) 先選本尊（困境對位）
  2) 再選方案
  3) 去綠界完成付款

## 2) 轉換漏斗（Conversion Funnel）
- 首頁：三秒內講清楚「你卡在哪 → 我給你哪條路 → 立刻預定」。
- 本尊頁：用「路徑宣言」+「方案卡（最推薦置頂）」+「透明流程」+「FAQ 拆疑」形成閉環。
- 付款頁（Pay Hub）：集中列出四尊方案與注意事項，全部導向綠界（第三方金流），本站避免蒐集敏感個資。

## 3) 資訊架構（IA）
- /：首頁（困境對位、四尊入口、流程、FAQ、社群信任）
- /puja（法會總覽）：四尊比較、各尊方案入口
- /pay：付款入口（綠界導流、注意事項）
- /deities/yellow（黃財神｜穩財）
- /deities/mahashri（守財與財庫）
- /deities/ganapati（清明決策）
- /deities/kurukulla（貴人與攝受）
- /sutra：經典主軸頁（以《金光明最勝王經》敘事）
- /about：關於與傳承立場
- /proof：見證與常見回饋（可持續擴充）
- /topics/:slug：SEO 長尾困境內容入口（導向對應本尊）
- /terms：條款/隱私

## 4) 視覺設計（我們將嚴格遵守）
**設計運動**：Lacquer Noir + Gold Foil Editorial（漆黑金箔的當代宗教編輯風）
- 深黑底、暖金光、紙張顆粒、金箔邊框、斜向構圖
- 大標題用「書法感 Display 字體」；內文用「耐讀襯線」
- 不走電商模板；要像高端長頁敘事（Sale Page）

## 5) SEO 原則（技術 + 內容）
- 每頁：獨立 title / description / OG / canonical
- 結構化資料：Organization、WebSite、BreadcrumbList、Product/Service（依頁面）
- 內部連結：
  - Topic → 對應本尊 → Pay
  - 本尊頁互相導覽（「你可能更適合…」）
- 內容策略：
  - 先做 8–12 篇困境型主題（可逐步擴充）

## 6) 合規與敏感性
- 不做醫療/投資報酬保證式承諾；改用「發心、護持、回向、因緣」語彙。
- 清楚標示：本站不在站內收集敏感個資；付款於第三方金流平台完成。
